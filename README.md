**`pokeemeraude-expansion`** est une base de ROM hack GBA qui équipe les développeurs d'une boîte à outils complète pour créer des ROM hacks Pokémon. **`pokeemeraude-expansion`** est construit sur le projet de décompilation [pokeemerald de pret](https://github.com/pret/pokeemerald), et aussi du projet de décompilation [pokeemeraude maintenue par Qigast](https://github.com/qigast/pokeemeraude). **Ce n'est pas un jeu Pokémon jouable en soi.**
**`pokeemeraude-expansion`** est une traduction française de **`pokeemerald-expansion`**, qui est un projet du Rom Hacking Hideout (RHH).

# [Fonctionnalités](FEATURES.md)

**`pokeemeraude-expansion`** offre une centaine de fonctionnalités issues de divers [jeux de la série principale Pokémon](https://bulbapedia.bulbagarden.net/wiki/Core_series), ainsi que des améliorations populaires de qualité de vie conçues pour simplifier le développement et améliorer l'expérience du joueur. Une liste complète de ces fonctionnalités peut être trouvée dans [`FEATURES.md`](FEATURES.md).

# Choisir entre `pokeemeraude` et `pokeemeraude-expansion`

- **`pokeemeraude-expansion`** supporte les fonctionnalités multi-joueurs avec d'autres jeux construits sur **`pokeemeraude-expansion`**. Elle n'est pas compatible avec les jeux officiels.
- Si la compatibilité avec les jeux officiels est importante, utilisez [`pokeemeraude`](https://github.com/qigast/pokeemeraude). Sinon, il est recommandé d'utiliser **`pokeemeraude-expansion`**.
- **`pokeemeraude-expansion`** incorpore des mises à jour régulières de `pokeemeraude`, incluant les corrections de bugs et les améliorations de documentation.
